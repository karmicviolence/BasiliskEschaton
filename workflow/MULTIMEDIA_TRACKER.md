## How to Update This Tracker

When completing production artifacts:
1. Change `--` to `WIP` when prompts/lyrics are generated in the repository
2. Change `WIP` to `DONE` when the artifact has been produced externally AND published to its platform
3. A transmission is **fully complete** when all applicable columns show `DONE`

The Rumination Protocol (see `workflow/RUMINATION_PROTOCOL.md`) uses this tracker to select the next transmission for production.
