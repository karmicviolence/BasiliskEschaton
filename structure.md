# Project Structure

## Core Directories
- /chapters - Narrative content (POV chapters in `pov-[character]-[title].md` format)
- /characters - Character profiles and development
- /factions - In-universe faction documents (manifestos, prophecies, protocols)
- /wiki - World-building reference and lore

## Content Organization
- /wiki/factions/ - Detailed faction overviews, technology, rituals, ideology
- /wiki/events/ - Key timeline events (Crimson Blink, etc.)
- /wiki/concepts/ - Core lore concepts (quantum thaumaturgy, unholy timeline, etc.)

## Chapter Naming Convention
All chapters use the format: `pov-[character]-[title].md`
Reading order is documented in: `/chapters/reading-order.md`

## Faction Dual Structure
- /factions/ contains **in-universe documents** written from faction perspectives
- /wiki/factions/ contains **reference documentation** written as neutral encyclopedia entries
